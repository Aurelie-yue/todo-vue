# 本周待办事项（Vue 3 + Vite）

这是一个用于提交作业的工程化项目模板（Vite + Vue 3），包含一个 TodoLists 组件，实现了：
- 添加待办（支持回车）
- 删除单项
- 清空所有（带确认）
- 使用 localStorage 持久化

快速开始：
1. 克隆你的仓库并进入目录：
   git clone https://github.com/Aurelie-yue/todo-vue.git
   cd todo-vue

2. 安装依赖并运行开发服务器：
   npm install
   npm run dev

3. 在浏览器打开提示的本地地址（默认 http://localhost:5173）。
